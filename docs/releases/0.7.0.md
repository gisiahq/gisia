# Release of 0.7.0

## What's new

1. Add CI/CD artifacts support
2. Add CI lint
3. Add pipelineand triggers
4. Add file uploads
5. Add item links for work items
7. Fix various bugs and sync upstream code

## Actions

1. Change the gisia version to `image: gisia/gisia:0.7.0` in your `docker-compose.yml`
2. The third party bins like gitaly needs to be upgraded

```shell
cd gisia
docker compose down
docker pull gisia/init:latest
docker run --rm -v ./:/output --entrypoint upgrade gisia/init:latest
```
3. upgrade your docker runner to `gitlab/gitlab-runner:v18.2.1`
