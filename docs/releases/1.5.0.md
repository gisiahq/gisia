# Release of 1.5.0

## What's new

1. Add group labels
2. Add expandable diff context on the diff page
3. Add review checkboxes to diffs
4. Restyle the comment form
5. Fix group members not showing in merge requests

## Actions

1. Change the gisia version to `image: gisia/gisia:1.5.0` in your `docker-compose.yml`
2. Run `docker compose up -d` to update the web service
