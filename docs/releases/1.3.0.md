# Release of 1.3.0

## What's new

1. Add search to issues, epics, and merge requests
2. Add mention notifications for notes (including on merge requests)
3. Add pin feature for namespaces
4. Add pagination for project lists
5. Let tags trigger pipelines
6. Show protected keywords on protected pipelines
7. Let admins block users
8. Fix turbo confirmation dialogs
9. Fix breadcrumb for projects
10. Fix paging params

## Actions

1. Change the gisia version to `image: gisia/gisia:1.3.0` in your `docker-compose.yml`
