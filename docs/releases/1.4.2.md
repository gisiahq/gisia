# Release of 1.4.2

## What's new

1. Add merge request reviews
2. Add comments API
3. Add notes API
4. Show durations on pipeline pages
5. Change the default email style

## Actions

1. Change the gisia version to `image: gisia/gisia:1.4.2` in your `docker-compose.yml`
2. Run `docker compose up -d` to update the web service
